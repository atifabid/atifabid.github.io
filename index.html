Prompt2CAD (previously text2CAD)

I have developed a text-to-CAD system that converts natural language prompt into parametric CAD models.

    This project involved:
  • Parsing natural language (prompts) into geometric primitives
  • Handling parametric constraints and dependencies
  • Generating executable CAD operation sequences via python script
  • Direct CAD generation in Solidworks software
      • The prototype built and tested : Text2CAD
