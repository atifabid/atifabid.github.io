Selected Projects

  • Prompt2CAD — Neuro-Symbolic Generative Design [Code] [Demo] LLM CAD
    Text-to-CAD pipeline that translates natural language into executable SolidWorks geometric feature trees, guaranteeing 100% geometric validity. Bridges LLM reasoning with physical design constraints.
  • Edge-Based Person Detection System [Demo] IoT Edge AI
    Real-time surveillance node for constrained IoT hardware. Optimized lightweight detection models for person and object detection in agricultural settings. Paper under submission.
  • Reproducibility Study: SCS-YOLO for Precision Agriculture [Code] Computer Vision PyTorch
    Re-implemented the SCS-YOLO architecture (originally in Computers and Electronics in Agriculture) on Google Colab. Validated on open-source pest/crop datasets, achieving mAP results consistent with the original authors.
  • Local LLM Inference Benchmarks [Code] [Data] LLM Benchmarking
    Rigorous benchmarking of Llama 3, Qwen, and gptoss20B on consumer hardware (RTX 40-series). Evaluated vLLM, Ollama, and TensorRT-LLM to identify optimal local inference setups.
  • Edge AI: Model Compression for Agriculture [Code] TFLite Compression
    3-stage compression pipeline (Pruning → Quantization → TFLite export) for RNN-based precision agriculture models, enabling real-time inference on legacy Android devices.
